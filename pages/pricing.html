FAQ

What counts as a request?

Each MCP tool call counts as one request. Initializing a session also counts as one request. Checking usage or health does not count.

When does my usage reset?

Usage resets on the 1st of each month (UTC).

What happens if I hit my limit?

You'll receive a clear error message with your current usage and remaining capacity. You can upgrade at any time — your key stays the same.

Which AI clients are supported?

Any MCP-compatible client that supports Streamable HTTP transport. This includes Claude Desktop, Cursor, Windsurf, VS Code with GitHub Copilot, and more. See the docs for setup instructions.

Can I upgrade or downgrade at any time?

Yes. Your API key stays the same when you change plans. Changes take effect immediately.

Email us at hello@ezbizservices.com. We typically respond within 24 hours on business days.
