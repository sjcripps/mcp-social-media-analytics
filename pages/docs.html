Documentation

Connection

Server URL: https://social.ezbizservices.com/mcp

Auth: Pass your API key as X-API-Key header or api_key query parameter.

Tools

analyze_profile
Analyze a social media profile or brand presence — posting patterns, content themes, audience indicators, and growth recommendations.

Parameters:
- username (string): Social media username or handle
- platform (string, optional): Platform (twitter, instagram, linkedin, facebook, tiktok)
- business_name (string, optional): Business name for broader search

score_engagement
Score social media engagement for a brand or topic — engagement rate estimates, content type effectiveness, posting time analysis, and benchmarks.

Parameters:
- brand_or_topic (string): Brand name or topic to analyze
- platform (string, optional): Platform to focus on (optional, analyzes all if omitted)

detect_trends
Detect trending topics and conversations in a niche — viral content patterns, emerging topics, sentiment shifts, and opportunity alerts.

Parameters:
- niche (string): Industry or niche to monitor
- timeframe (string, optional): Timeframe: today, this_week, this_month

research_hashtags
Research effective hashtags for a topic — popularity estimates, related hashtags, niche vs broad classification, and recommended hashtag sets.

Parameters:
- topic (string): Topic or keyword for hashtag research
- platform (string, optional): Target platform (instagram, twitter, tiktok, linkedin)
- count (number, optional): Number of hashtags to return (default 20)

API Endpoints

  • POST /mcp — MCP protocol endpoint
  • POST /api/keys/signup — Create free API key (body: {name, email})
  • GET /api/keys/usage?key=YOUR_KEY — Check usage
  • GET /api/pricing — View pricing tiers
  • GET /health — Health check
